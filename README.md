# Garden-Of-Grind
A drop-in helper mod for the garden of grind challenge run for GTNH.

If using this, make no other gog config changes. You do not need to modify Thaumcraft.cfg, Witchery.cfg, GregTech.cfg, or Hodgepodge. All changes are done internally by this mod.
